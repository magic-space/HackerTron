Spiel Aktualisierungen
0.33: kleinere Änderungen.
  • Verbesserungen der Benutzeroberfläche.
    • Das kleine Zahnradmenü wurde entfernt.
    • Das Dropdown-Menü "Mehr" enthält alle Optionen aus dem Zahnradmenü.
    • Größere Befehlseingabe.
    • Kleinere Textkorrekturen.
  • Einige Fehler wurden behoben.
    • Autoscroll behoben, es sollte jetzt richtig funktionieren.
    • Fast-Tab-Switch mit den Verknüpfungen behoben.
    • Fehler bei den Errungenschafgen achievement wurde behoben, man hatte diese schon beim Akkulevel 1 bekommen obwohl sie erst ab Level 5 erreicht wird..
    • Reduzierte Spiel-FPS (von 30 auf 20).
0.32: Akkusystem (Testen der Implementierung des Mikromanagements).
  • Akku-Mikromanagementsystem hinzugefügt.
    • Bewege den Mauszeiger über den Akku, um ihn aufzuladen oder tippe mit dem Finger drauf.
    • Upgrade your battery to increase capacity and charge time.
    • Akkueffekt erhöhen Erf. Multi. um 25%, Geld Multi. um 25%, Zeit Multi. um 10%..
  • Improved terminal experience.
    • Autocomplete command when pressing TAB key (feature in beta).
    • Retrieve lastest commands sent with up/down arrow keys.
    • CTRL+L clear terminal logs.
    • Copy/paste enabled.
  • Commands improvements.
    • Added -q argument for multibuy.
    • -q must be followed by an amount (in numbers) or the max keyword.
    • Added achievements command.
  • Better balancing.
    • Scripts: prices, exp and money reward changed.
    • Autoscript: prices changed.
    • Servers: reduced web server effects.
  • Improved UI.
    • Added tooltips on important gameplay elements.
    • Better navigation tab.
    • Switch tabs by pressing SHIFT+left/right arrow keys.
  • Many bugs fixes.
0.31: first beta-release.
  • Initial beta-release. Expect a buggy game with few gameplay features.
  • Thanks for taking part of the game development.
